Task 21

Контекстный селектор

1. Сделайте селектор, который выберет все абзацы p.

p { color: red; }

2. Сделайте селектор, который выберет все абзацы p внутри дивов div.

div p { color: red; }

3. Сделайте селектор, который выберет все h2 внутри дивов div.

div h2 { color: red; }

Группировка селекторов

4. Сделайте селектор, который выберет абзацы p и h2 одновременно.

p, h2 { color: red; }

5. Сделайте селектор, который выберет все h2 внутри дивов div и одновременно все абзацы p.

div h2, p { color: red; }

6. Сделайте селектор, который выберет все h2 внутри дивов div и одновременно все абзацы p внутри дивов div.

div h2, div p { color: red; }

На id

7. Сделайте селектор, который выберет все элементы из id = test.

#test { color: red; }

8. Сделайте селектор, который выберет все абзацы p из id = test.

#test p { color: red; }

9. Сделайте селектор, который выберет все абзацы p из id = test и одновременно все h2.

#test p, h2 { color: red; }

На классы

10. Сделайте селектор, который выберет все элементы с классом test.

.test { color: red; }

11. Сделайте селектор, который выберет все абзацы p с классом test.

.test p { color: red; }

12. Сделайте селектор, который выберет все элементы с классом test и одновременно все элементы с классом hello.

.test, .hello { color: red; }

13. Сделайте селектор, который выберет все абзацы p с классом test и одновременно все h2 с классом hello.

p.test, h2.hello { color: red; }

14. Сделайте селектор, который выберет все абзацы p с классом test и одновременно все элементы с классом hello.

p.test, .hello { color: red; }

На id и классы вместе

15. Сделайте селектор, который выберет все абзацы p с классом test из дива div с id = hello.

div#hello p.test { color: red; }

16. Сделайте селектор, который выберет все абзацы p с классом test из элементов с классом hello.

.hello p.test { color: red; }

17. Сделайте селектор, который выберет все абзацы p с классом test из дивов div с классом hello.

div.hello p.test { color: red; }

18. Сделайте селектор, который выберет все h2 из класса test.

.test h2 { color: red; }

На мультиклассы

19. Сделайте селектор, который выберет все элементы, которые имеют одновременно 2 класса: test и hello.

.test.hello { color: red; }

20. Сделайте селектор, который выберет все абзацы p, которые имеют одновременно 2 класса: test и hello.

p.test.hello { color: red; }

21. Сделайте селектор, который выберет все h2, которые имеют одновременно 3 класса: test, hello и eee.

h2.test.hello.eee { color: red; }

Дочерний селектор >

22. Сделайте селектор, который выберет все i, которые лежат непосредственно внутри p.

p > i { color: red; }

23. Сделайте селектор, который выберет все li, которые лежат непосредственно внутри ul.

ul > li { color: red; }

24. Сделайте селектор, который выберет все a, которые лежат непосредственно внутри p и одновременно все a с классом test.

p > a, a.test { color: red; }

Соседний селектор +

25. Сделайте селектор, который выберет все i, которые лежат непосредственно после b.

b + i { color: red; }

26. Сделайте селектор, который выберет все i, которые лежат непосредственно после b из элемента с id = test.

#test b + i { color: red; }

27. Сделайте каждый абзац, следующий сразу после элемента ul, красного цвета.

ul + p { color: red; }

Родственный селектор ~

28. Сделайте селектор, который выберет все абзацы p идущие после h1.

h1 ~ p { color: red; }

29. Сделайте селектор, который выберет все абзацы p идущие после h2 из элемента с id = test.

#test h2 ~ p { color: red; }

Универсальный селектор *

30. Сделайте селектор, который выберет все элементы на странице

* { color: red; }

31. Сделайте селектор, который выберет все i, лежащие внутри любых элементов, лежащих внутри абзаца p.

p * i { color: red; }

Селекторы атрибутов

32. Сделайте селектор, который выберет все элементы с атрибутом title

*[title] { color: red; }

33. Сделайте селектор, который выберет все абзацы p с атрибутом title

p[title] { color: red; }

34. Сделайте селектор, который выберет все абзацы p с атрибутом title, который равен "привет".

p[title="привет"] { color: red; }

35. Сделайте селектор, который выберет все ссылки a с атрибутом href, значение которого начинается на "http://".

a[href^="http://"] { color: red; }

36. Сделайте селектор, который выберет все ссылки a с атрибутом href, значение которого заканчивается на ".html".

a[href$=".html"] { color: red; }

37. Сделайте селектор, который выберет все абзацы p с атрибутом title, в значении которого есть слово "привет".

p[title*="привет"] { color: red; }

38. Сделайте селектор, который выберет все абзацы p с классом, который начинается на "top-".

p[class|="top"] { color: red; }
